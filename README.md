## Présentation

Ce dépôt regroupe l'ensemble des projets réalisés durant la **Phase 2** du parcours **Fullstack** proposé par **Akieni Academy** (Cohorde 2 — 2026).

Cette phase est consacrée au renforcement des **fondamentaux du développement web front-end** à travers la conception de pages web structurées, stylisées et entièrement responsives. Chaque semaine propose **trois projets** classés par niveau de difficulté :

| Niveau | Description |
|:---:|:---|
| **Débutant** | Exercices guidés |
| **Intermédiaire** | Projets encadrés |
| **Avancé** | Défis autonomes |

---

## Objectifs pédagogiques

À l'issue de cette phase, les compétences suivantes sont acquises :

- Maîtriser la **structure HTML sémantique** et accessible
- Comprendre le **modèle de boîtes CSS** (Box Model), la cascade et la spécificité
- Concevoir des **interfaces web complètes** et responsives
- Utiliser **Git & GitHub** comme outils de versionning et de collaboration
- Adopter de **bonnes pratiques** de développement (conventions, organisation, lisibilité)

## Compétences développées

<details>
<summary><strong>Voir la liste complète</strong></summary>

### HTML
- Structure sémantique (`<header>`, `<nav>`, `<main>`, `<section>`, `<article>`, `<footer>`)
- Formulaires et tableaux
- Médias (images, audio, vidéo)
- Accessibilité de base (attributs ARIA, `alt`, `lang`)

### CSS
- Box Model (marges, bordures, padding, contenu)
- Sélecteurs CSS (classes, ID, attributs, combinators)
- Pseudo-classes et pseudo-éléments
- Variables CSS (Custom Properties)
- Cascade, spécificité et héritage
- Flexbox et Grid (introduits en S4)

### Outils & Méthodologie
- Workflow Git (init, add, commit, push, pull, branches)
- Organisation de projets en dossiers
- Conventions de nommage
- Débogage avec les DevTools du navigateur

## Exécution locale

Pour visualiser les projets en local :

```bash
# 1. Cloner le dépôt
git clone https://github.com/FullStack-Steven/Akieni-Phase2.git

# 2. Se déplacer dans le répertoire
cd Akieni-Phase2

# 3. Ouvrir le projet souhaité dans un navigateur
#    Exemple pour le projet 2.1 :
open S2/Projet-2.1/index.html
```

> *Sur Linux, remplacez `open` par `xdg-open`. Sur Windows, double-cliquez simplement sur le fichier `index.html`.*
